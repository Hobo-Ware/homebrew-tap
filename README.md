# Hobo-Ware tap

Homebrew tap for [Hobo-Ware](https://github.com/Hobo-Ware) tools. Tools for the discerning degenerate.

## stdusk

A native Rust quake terminal with a real GUI tab bar and ambient AI-CLI awareness.

```sh
brew install hobo-ware/tap/stdusk
```

Then hit `Ctrl+\`` to summon it. See [Hobo-Ware/stdusk](https://github.com/Hobo-Ware/stdusk).
